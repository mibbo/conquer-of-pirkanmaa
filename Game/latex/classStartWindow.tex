\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classStartWindow}{Start\+Window}} class is a custom Qdialog that pops a starting window when starting the game. 

In starting window players can set their names and colors. Also the map size form 3 options (small/normal/huge).

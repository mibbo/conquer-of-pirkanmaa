\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classStudent_1_1Warrior}{Warrior}} class represents a worker that can be used to attack the enemy Head\+Quarter-\/building. 

\mbox{\hyperlink{classStudent_1_1Warrior}{Warrior}} has 5 hit points and can attack the other players warriors by moving to the same tile with them. \mbox{\hyperlink{classStudent_1_1Player}{Player}} wins the game when his/her warrior moves to the enemys headquarter
